Para verificar las cuentas nuevas requiere de una cuenta y credenciales de paginas de emails como Mailtrap.
Para iniciar servidor php dentro de la carpeta api/public: php -S localhost:3000.
Para iniciar el proyecto react dentro de la carpeta WEB: npm run dev / pnpm run dev.
Las contraseñas de todos los usuarios pre-cargados son 'prueba'.
